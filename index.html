CSS code Description Selector Declaration
Property Value
th{color:blue;} changes the color of the text in the headings to blue th color blue
table{border-color:blue;} changes the color of the borders to blue table border-color blue
table{border:1px solid black;} creates a solid black border with thickness of 1px around the table table border 1px solid black
table{text-align: left;} Moves all heading texts in the table to the left, and also other texts if not specified table text allignment left
table{border-collapse: collapse;} This will make the borders collapse into a single border and so eliminate double borders table border-collapse collapse
td {background-color: yellow;} This will make the background color of the cells, apart from the heading, to turn yellow. td background-color yellow
table {background-color: yellow;} This will make the background color of all the cells to turn yellow. table background-color yellow
td {border-radius: 10px;} This will make the corners of the cells to be rounded instead of pointed.
Note: this will not work with border-collapse in place. 		td border-radius 10px
th {border-style:dotted; } This will make the lines of the border around the headings to be dotted. Other styles include dotted
dashed, solid, double, groove, ridge, inset, outset,none.		 th border-style dotted
